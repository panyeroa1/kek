
# APP OVERVIEW: KEK Messenger

KEK is an advanced AI messenger designed for product hunters, drop-shippers, and tech enthusiasts. It bridges the gap between AI reasoning and real-world market data.

## Features Implemented
- **AI Chat Interface**: Mobile-first messenger UI with multi-model support.
- **Fast Mode**: Uses `gemini-2.5-flash-lite` for instant queries.
- **Research Grounding**: Google Search grounding via `gemini-3-flash-preview` for viral TikTok/Alibaba product trends.
- **Location Research**: Google Maps grounding via `gemini-2.5-flash` for logistics and local sourcing.
- **Visual Intelligence**: Image analysis via `gemini-3-pro-preview` for identifying products from photos.
- **Deep Thinking**: Complex reasoning mode with a 32,768 token thinking budget using `gemini-3-pro-preview`.
- **Live Voice**: Real-time voice interaction using `gemini-2.5-flash-native-audio-preview-12-2025`.
- **Product Ecosystem**: Database of apps connecting to hardware (e.g., Headphone AI Translators).

## Future Roadmap
- Integration with external store APIs (Alibaba SDK).
- Multi-user collaborative research rooms.
- Real-time price tracking graphs.
